# Recipe Manager Frontend

Modern React application for managing personal recipes.

## Features
- User authentication
- Create, edit, delete recipes
- Search recipes
- Filter by category
- Mark favorites
- Rate recipes
- Responsive design
- Modern UI with animations

## Tech Stack
- React 18
- Vite
- React Router
- Axios
- Bootstrap 5

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update API URL in `src/api/axios.js`:
```javascript
baseURL: 'http://localhost:5000'
```

3. Start development server:
```bash
npm run dev
```

4. Open browser to `http://localhost:3000`

## Build for Production

```bash
npm run build
```
